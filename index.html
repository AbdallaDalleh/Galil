Galil-3-0

ASYN based EPICS driver for Galil products

Download this project as a .zip file Download this project as a tar.gz file

Welcome

This software branch was sent to Galil for their own separate development on 20/11/2013. The driver advertised and supported by Galil should now be considered a separate branch and it's origins are from this software.

Since that time, this original branch has advanced significantly

The driver has the following features:

A) Asyn model 3 architecture
B) Motor record support
C) Analog and Digital IO
D) Output TTL pulses for closed-loop motor delta distance (ie. output compare)
E) Time based profile motion (Trajectories)
F) Coordinate system axes (eg. Slit width)
G) Deferred moves facility
H) Coordinated motion for up to 8 motors
I) Sophisticated connection management
J) Fast update rates 8ms
K) Example MEDM screens
L) Example IOC
M) Autosave request files
N) RIO PLC supported
O) Intended to replace Galil EPICS 1-5 driver

The driver is for linux runlevel 5, or windows.

All motion modes specified work on the 21x3 or newer motion controller.

I hope it is useful :)
