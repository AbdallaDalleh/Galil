Galil-3-1

ASYN based EPICS driver for Galil products

Download this project as a .zip file Download this project as a tar.gz file

Welcome

This software branch was sent to Galil for their own separate development on 20/11/2013. The EPICS driver advertised and supported by Galil should now be considered a separate branch and it's origins are from this software.

Since that time, this original branch has advanced significantly

This driver has the following features:

A) Asyn model 3 architecture
B) Motor record support
C) Motor record PREM/POST function is supported
D) Analog and Digital IO
E) Output TTL pulses for closed-loop motor delta distance (ie. output compare)
F) Time based profile motion (Trajectories)
G) Coordinate system axes (eg. Slit width)
H) Kinematics adjustable via database puts
I) Deferred moves facility
J) True coordinated motion for up to 8 motors
K) Auto motor power On/Off with adjustable delays
L) Sophisticated Galil code generator, no need to program low level
M) Ability to construct low level code from templates
N) User definable control/monitor functions with full record support
O) Sophisticated connection management
P) Fast update rates 8ms
Q) Example MEDM screens
R) Example IOC
S) Autosave request files
T) RIO PLC supported
U) Intended to replace Galil EPICS 1-5 driver

Older versions can be found here:

https://github.com/motorapp/Galil-3-0/releases

All motion modes specified work on the 21x3 controller or newer.

I hope you find this software useful :)
